### 数据模型变更

#### exercises 结构

```json
// 旧
{ "weight": "80", "reps": "10", "done": true }

// 新
{ "weight": "80", "reps": "10", "status": "done" }
```

**三态：** `pending`（未开始）| `done`（已完成）| `skipped`（跳过）

skipped 的组 weight/reps 保留当前输入值（不清空），方便补做时沿用。

#### templates 结构

```json
// 旧
{ "name": "卧推", "sets": 4, "reps": 10 }

// 新
{ "name": "卧推", "sets": 4, "reps": 10, "restSeconds": 90 }
```

`restSeconds` 可选，默认 60。训练时倒计时自动使用该值。

### 进度条视觉

| 状态 | 颜色 | 效果 |
|------|------|------|
| pending | #2a2a2a | 空心 |
| done | #4ade80 | 实心 |
| 当前组 | #4ade80 | 呼吸脉动 |
| skipped | 斜线纹理 | `repeating-linear-gradient` 绿色斜线 |
| 动作间分隔 | #333 | 竖线 2px |

### 进度条可点击

- 点击**已完成动作（至少有一组 done）**的进度条段落 → 弹出半屏面板
- 面板显示该动作所有组：组号 | 重量 | 次数 | 状态(done/skipped/pending)
- 默认查看模式，每行右侧 ✎ 图标
- 点 ✎ → 该行进入编辑模式，可改重量/次数，面板顶部出现「取消」
- 改完自动保存

### 跳过操作（显式按钮）

| | 跳过这组 | 跳过这个动作 |
|------|------|------|
| 位置 | 完成按钮下方 | 动作名下方 |
| 字号 | 13px | 11px |
| 颜色 | #555 | #444 |
| 确认 | 无需确认 | 二次确认弹窗 |
| 行为 | status → skipped，直接推进 | 所有未完成组 → skipped，跳到过渡页 |

**跳过动作确认弹窗：**

```
┌──────────────────────────┐
│  跳过「卧推」？           │
│  剩余 2 组将标记为跳过    │
│                          │
│  [ 取消 ]  [ 确认跳过 ]  │
└──────────────────────────┘
```

**skip 时不修改 weight/reps**——保留当前值，补做时可以直接用。

### 结束页逻辑

```
训练结束（全部完成或用户点结束训练）
  │
  ▼
检查 exercises 中是否有 status === "skipped"
  │
  ├─ 有 ──→ 显示结束页（带补做入口）
  │          ├─ [补做 N 组] → enterMakeupMode()
  │          └─ [结束训练并保存] → 写入 localStorage，清除 CURRENT_KEY
  │
  └─ 无 ──→ 全部 done，直接保存（无确认）
```

### 补做模式

**进入：** 从结束页点 [补做 N 组]

**流程：**
```
1. 收集所有 status === "skipped" 的组 → makeupQueue: [{exIdx, setIdx}, ...]
2. makeupMode = true
3. 从第一个 skipped 组开始，和正常大卡片一样
4. 每完成一组：status → "done"，推进到下一个 skipped 组
5. 补做时不写 CURRENT_KEY（不触发断点保护）
6. 全部补做完 → 回到结束页，重新计算 skipped 数量
```

**补做时的卡片的特殊处理：**
- 动作名处显示「补做模式」
- weight 保留 skip 前的值（如果有填过），否则空
- 点击完成 → 正常倒计时 → 推进

**退出补做：** 补做页面底部有「回到总结」按钮，未完成的保持 skipped

### 对比现状

| 维度 | 原来 | 改后 |
|------|------|------|
| 每组操作 | 找 → 填 → 勾（3N 次点击） | 点[完成]（N 次点击） |
| 跳过组 | 无 | 点「跳过这组」→ 标记 skipped |

| 进度条 | 纯展示 | 三色 + 可点击编辑 |
| 组间休息 | 固定选项 | 模板预设 + ±15s 微调 |
